## Por que foi construído
Este projeto foi desenvolvido para colocar em prática e aprimorar minhas habilidades no desenvolvimento de sistemas web, utilizando tecnologias comuns do mercado.

Além disso, o site também serve como um exemplo de como transformar solicitações de clientes (briefings) em uma solução funcional, clara e alinhada às necessidades do negócio.
